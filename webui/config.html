⚡ Vision Fallback
When enabled, images are intercepted and sent to a dedicated vision model instead of the main chat model.
Vision Model
Vision models: gemini-3-flash-preview, llama3.2-vision, gpt-4o-mini
💡 If left empty, the plugin will automatically use the key from your Agent Zero .env file (e.g. API_KEY_OLLAMA_CLOUD for Ollama Cloud).
